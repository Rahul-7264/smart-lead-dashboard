# Smart Leads Dashboard

A full-stack Lead Management Dashboard built using React, TypeScript, Node.js, Express, MongoDB Atlas, and Tailwind CSS.

---

# Features

- Add Leads
- Edit Leads
- Delete Leads
- Search Leads
- Filter by Status
- Lead Source Tracking
- Dashboard Analytics
- Pagination
- CSV Export
- Dark Mode
- Responsive UI

---

# Tech Stack

## Frontend
- React
- TypeScript
- Tailwind CSS
- Axios
- Vite

## Backend
- Node.js
- Express.js
- TypeScript
- MongoDB Atlas
- Mongoose

---

# Installation

## Clone Repository

```bash
git clone <your-github-link>
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# API Endpoints

## Get All Leads

```http
GET /api/leads
```

## Add Lead

```http
POST /api/leads
```

## Update Lead

```http
PUT /api/leads/:id
```

## Delete Lead

```http
DELETE /api/leads/:id
```
